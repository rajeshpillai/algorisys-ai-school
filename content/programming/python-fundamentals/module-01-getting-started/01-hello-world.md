## Slide

# Hello, World!

Every programming journey begins with a simple greeting. In Python, displaying text on screen is remarkably simple.

### The `print()` function

Python's `print()` function outputs text to the console:

```python
print("Hello, World!")
```

Key points:
- `print()` is a built-in function — no imports needed
- Text must be enclosed in quotes (single `'` or double `"`)
- Python executes code line by line, top to bottom

### Why start here?

The "Hello, World!" program is a tradition in computer science. It verifies that:
1. Your environment is set up correctly
2. You can write and run code
3. You understand the basic syntax

## Discussion

Let's explore why Python is one of the most popular languages for beginners. What makes `print("Hello, World!")` in Python simpler than the same program in languages like Java or C++? Think about the amount of boilerplate code needed.
